Add / Remove Domain to / from Black Listing

You can use Add/Remove Domain to customize your black lists, by adding a specific domain to one category or remove it from all categories.

Note: right now update will overwrite your customized filters with the default categories.

Add new Domain

Click on Add/Remove Domain

Write required domain to add on Domain Name: field

Select Category from Category:

Select Add from Action:

Submit.





Remove Domain

Click on Add/Remove Domain

Write required domain to add on Domain Name: field

Select Remove from Action:

Submit.




Enable / Disable Categories.

To Enable to Disable categories simply

Click on Enable/Disable Category

Select All Required Categories to Block from All Categories: Selected Categories will Be Highlighted.
